## ICT235 Data Structures and Algorithms 

These past year paper solutions cater to Mathematics and Programming Academic Units at SUSS.  

Starting in 2025S1, ICT231 Data Structures and Algorithms 1 and ICT232 ICT232 Data Structures and Algorithms 2 have been reorganised into
* ICT235 Data Structures and Algorithms
* ICT325 Algorithm Design and Analysis

If you need tuition, feel free to contact me at maths_and_programming@outlook.sg or @geodome on Telegram.  

ICT235 will provide a comprehensive overview of fundamental data structures and algorithms essential for ICT and software engineering. 
It is designed to deepen understanding of data organisation, storage, and retrieval, alongside efficient algorithm design and analysis. 
The course is structured to build from basic concepts to more complex structures and algorithms using Python, ensuring a solid foundation 
for further study and practical application.

Topics
* Algorithm Analysis
* Recursion
* Array-based Sequences
* Stacks and Queues
* Linked List and Positional List
* Tree and Tree Traversal Algorithms
* Priority Queues
* Maps
* Hash
* Skip Lists
* Sorting
* Selection
