## MTD315 Computer Interactive Graphics

This course provides a basic background on geometry and computer graphics for virtual reality systems.

### Topics

* Introduction to Graphics Systems and Models
* Basic OpenGL
* Simple programs in 2D and 3D
* Three Dimensional Graphics
* Geometry and transformations
* Implementation
* Polygon fill
* Discrete methods
* Bitmaps and pixel maps
* Compositing and transparency
* Modelling
* Tree structured models
