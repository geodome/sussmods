## ICT231 Data Structures and Algorithms I.

Prior to 2024S1, ICT231 was previously known as MTH251.

These past year paper solutions cater to Mathematics and Programming Academic Units at SUSS.  

If you need tuition, feel free to contact me at maths_and_programming@outlook.sg.  

ICT231 Data Structures and Algorithms I will provide students with an understanding of the  common algorithms and data structures used in information technology.  

The topics covered are of central importance for many applications in data analytics and information technology.  

The course gives a comprehensive introduction to algorithm analysis, basic data types such as stacks, queues and trees and how these data types are implemented with linked lists.

Codes will be written and presented in Python.
