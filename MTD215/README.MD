## MTD215 Application of C++ in Multimedia

These past year paper solutions cater to Mathematics and Programming Academic Units at SUSS.  

If you need tuition, feel free to contact me at maths_and_programming@outlook.sg.  

The course provides greater programming exposure to multimedia students who have completed ICT131, as an alternative to MTD213.  

The course briefly covers the basics of the C++ language and focuses on the application of C++ for animation and games design.  

Students will learn to create objects  and to animate them using C++.  

In addition they will acquire the skills required to refine existing C++ codes in order to modify existing animation and games software.
